Описание
Графическое приложение на Python (`tkinter`) для хранения и управления личной коллекцией фильмов.  
Поддерживает добавление фильмов, фильтрацию по жанру и году, а также сохранение/загрузку данных в формате JSON.

Функционал:
- Форма ввода с полями: Название, Жанр, Год выпуска, Рейтинг
- Табличный вывод списка фильмов
- Фильтрация по жанру (выпадающий список) и году выпуска (точный поиск)
- Валидация ввода (год — число, рейтинг — диапазон 0–10)
- Сохранение и загрузка данных через `movies.json`
- Интеграция с Git (`.gitignore`, коммиты)

Требования
- Python 3.7+
- Стандартные библиотеки: `tkinter`, `json`, `os`
